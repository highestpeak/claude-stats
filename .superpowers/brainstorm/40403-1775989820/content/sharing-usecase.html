分享这个 dashboard 的方式是？

不同的分享场景会影响整体设计方向

A

截图 / 导出图片

截图发给朋友，或导出为 PNG/PDF。不需要对方有访问权限，简单直接。

B

本地访问 + 分享链接

把 dashboard 部署到本机，局域网内同事可以访问。或者用 ngrok/cloudflare tunnel 临时暴露给朋友看。

C

可嵌入的 Stats Card

类似 GitHub README stats card，生成一张可以放在 README / 个人主页的动态图片或 SVG。

D

以上都要（多种方式）

视场景选择，有时截图，有时让朋友直接来看，有时放 README。
